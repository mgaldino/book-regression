Introdução à Regresssão para Ciências Sociais

Capítulo 16 - Experimentos Naturais

Nós apresentamos no capítulo 6 o modelo de resultados potenciais para definir causalidade. Contudo, avançamos pouco sobre como na prática, fazemos para estimar efeitos causais com regressão, para além da estratégia de controla para variáveis de confusão. O objetivo do presente capítulo é apresentar, brevemente, três das técnicas principais para inferTência causal com dados observacionais, também chamados de experimentos naturais.
